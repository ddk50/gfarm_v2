gfcd

Name

gfcd -- Gfarm 作業ディレクトリを変更する。

Synopsis

gfcd [ディレクトリ]

DESCRIPTION

Gfarm 作業ディレクトリを 指定したディレクトリに変更します。 ディレクトリを指定しない場合は、ユーザの Gfarm ホームディレクトリである gfarm:/グローバルユーザ に変更します。

各Gfarm コマンドが作業ディレクトリを参照できるように、各ユーザの シェル設定ファイルに以下の定義を追加してください。

Bourne shell 系シェルの場合
gfcd() { eval "`gfsetdir $1`"; }
csh 系シェルの場合
alias gfcd 'eval `gfsetdir \!*`'

SEE ALSO

gfpwd(1), gfsetdir(1)
