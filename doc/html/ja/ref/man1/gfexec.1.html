gfexec

Name

gfexec -- Gfarm ファイルシステムのプログラムを実行する

Synopsis

gfexec [-N ノード総数] [-I ノード番号] [-s] プログラム名 [引数...]

DESCRIPTION

Gfarm ファイルシステムのプログラムを実行します．本コマンドはファ イルシステムノード上で実行する必要があります．

OPTIONS

-N ノード数

呼び出すプログラムのノード総数を変更します．

-I ノード番号

呼び出すプログラムのノード番号を変更します．

-s

「-N 1 -I 0」と指定したのと等価です．

-?

引数オプションを表示します．
