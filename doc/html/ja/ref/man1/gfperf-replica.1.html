Name

gfperf-replica — Gfarm ファイル複製の性能測定を行う。

Synopsis

gfperf-replica -s, --src 複製元gfsd名 -s, --dst 複製先gfsd名 [-l, --filesize ファイルサイズ] [-t, --testdir テストディレクトリ]

DESCRIPTION

引数 テストディレクトリ にテスト用のファイルを作成し、その複製性能を測定します。

OPTIONS

-s, --src 複製元gfsd名

複製元gfsd名を指定します。

-d, --dst 複製先gfsd名

複製先gfsd名を指定します。

-l, --filesize ファイルサイズ

テストで使用するファイルサイズを指定します。 デフォルトは1Mです。

-t, --testdir テストディレクトリ

テストを実施するディレクトリを指定します。 gfarm url (gfarm:///で始まるもの) を指定します。 デフォルトはgfarm:///tmpです。
