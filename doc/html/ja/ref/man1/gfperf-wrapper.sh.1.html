Name

gfperf-wrapper.sh — Gfarm 指定された認証方式でコマンド起動を行う。

Synopsis

gfperf-wrapper.sh sharedsecret | gsi_auth | gsi コマンド

DESCRIPTION

sharedsecret、gsi_authまたは、gsi認証のいずれかでコマンドを起動します。 予めgfmd及びgfsdに該当認証方式を受け入れる設定がなされている必要があります。

本コマンドは内部でGFARM_CONFIG_FILE環境変数を使用します。既にセットされている場合は本コマンドにより上書きされます。

OPTIONS

sharedsecret | gsi_auth | gsi

認証方式を指定します。
