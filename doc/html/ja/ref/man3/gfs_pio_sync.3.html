gfs_pio_sync

Name

gfs_pio_sync -- メモリ上のファイルの内容をディスクに反映させる

Synopsis

#include <gfarm/gfarm.h>

char *gfs_pio_sync(GFS_File gf);

DESCRIPTION

書き込み用にオープンしているファイル gf の、メモリ上にあるデータを実際にディスクに書き込み， 入出力の完了を待ってから戻ります．また， ファイルのメタデータも更新します． 別の関数 gfs_pio_datasync もメモリ上のファイルデータをディスクに書き込みますが， メタデータは更新しません．

RETURN VALUES

NULL

正常終了を示します．

GFARM_ERR_NO_MEMORY

メモリ不足のため処理を中止しました．

その他

上記以外のエラーが生じたことを示します．
