gfs_pio_getc

Name

gfs_pio_getc -- 一文字読み込み

Synopsis

#include <gfarm/gfarm.h>

int gfs_pio_getc(GFS_File gf);

DESCRIPTION

ファイル gf から一文字読み込みます。

RETURN VALUES

EOF

ファイルの末尾に達した場合、あるいはエラーが生じた場合、<stdio.h> で 定義されている定数 EOF を返します。 ファイル末尾なのか、エラーなのかは、関数 gfs_pio_error(3) で知る ことができます。

その他

ファイルから読み込んだ文字を返します。
