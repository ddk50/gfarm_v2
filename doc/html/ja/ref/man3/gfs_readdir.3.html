gfs_readdir

Name

gfs_readdir -- ディレクトリ内容読み込み

Synopsis

#include <gfarm/gfarm.h>

char *gfs_readdir(GFS_Dir dir, struct gfs_dirent ** entryp);

DESCRIPTION

引数 dir で指定したディレクトリから、1 つのディレクトリ・エントリ を読み込み、そのポインタを entryp の指す先に返します。 entryp の指す領域は、次に同じ dir 引数で gfs_readdir() ないし gfs_closedir() を呼び出した時に失われます。

RETURN VALUES

NULL

正常終了を示します。

その他

上記以外のエラーが生じたことを示します。

SEE ALSO

gfs_closedir(3), gfs_opendir(3), gfs_readdir(3)
