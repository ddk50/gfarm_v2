gfs_pio_set_local

Name

gfs_pio_set_local -- ローカルファイルビューの初期化

Synopsis

#include <gfarm/gfarm.h>

char *gfs_pio_set_local(int node, int nnode);

DESCRIPTION

ローカルファイルビューを利用する並列プロセスは、その処理の最初に、 そのノードのノード番号を node に、並列プロセスを構成する総ノード数を nnode に与えて、この関数を呼び出します。

RETURN VALUES

NULL

正常終了を示します。

その他

上記以外のエラーが生じたことを示します。

gfs_pio_set_view_local(3), gfs_pio_set_view_index(3)
