gfarm_url_fragments_replicate

Name

gfarm_url_fragments_replicate -- ファイルのレプリカを作成する

Synopsis

#include <gfarm/gfarm.h>

char *gfarm_url_fragments_replicate(const char *gfarm_url, int ndsthosts, char **dsthosts);

DESCRIPTION

ファイル gfarm_url の複製を、長さ ndsthosts の文字列配列 dsthosts で指定したホストに作成します。 この関数では、実行形式のファイルの複製には利用できません。

ファイルのファイルインデックスの総数より、指定されたホストの総数 ndsthosts が少ない場合は、指定されたホストがラウンドロビンで割り当てられます。

RETURN VALUES

NULL

正常終了を示します。

GFARM_ERR_NO_MEMORY

メモリ不足のため処理を中止しました。

GFARM_ERR_OPERATION_NOT_PERMITTED

引数として、実行形式のファイル、 ディレクトリなど通常ファイル以外のファイルを指定しました。

その他

上記以外のエラーが生じたことを示します。
