Name

gfs_acl_free — release a Gfarm ACL structure memory

Synopsis

#include <gfarm/gfarm.h>
gfarm_error_t gfs_acl_free( acl);  
gfarm_acl_t acl;
 

INPUT ARGUMENTS

acl

An ACL structure type to free the memory space.

RETURN VALUES

GFARM_ERR_NO_ERROR

The function terminated successfully.

Others

An error except the above occurred. The reason is shown by gfarm_error_string(3).
