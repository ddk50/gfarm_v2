Name

gfs_acl_get_tag_type — get the tag type from a Gfarm ACL entry

Synopsis

#include <gfarm/gfarm.h>
gfarm_error_t gfs_acl_get_tag_type( entry_d,  
  tag_type_p);  
gfarm_acl_entry_t entry_d;
gfarm_acl_tag_t *tag_type_p;
 

INPUT ARGUMENTS

entry_d

An ACL entry.

tag_type_p

A pointer to get the ACL tag type.

RETURN VALUES

GFARM_ERR_NO_ERROR

The function terminated successfully.

Others

An error except the above occurred. The reason is shown by gfarm_error_string(3).
